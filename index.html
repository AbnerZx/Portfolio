UI/UX Design

Um profissional de UI UX designer é responsável por criar uma experiência digital satisfatória e significativa para o usuário, combinando os princípios de design de interface do usuário (UI) e design de experiência do usuário (UX). Isso envolve a criação de produtos digitais que sejam atraentes, fáceis de usar e atendam às necessidades do usuário.



Web Design

Uma página da internet - ou website - é desenvolvida pelo web designer. Esse profissional é responsável tanto pelo projeto estético de um site quanto por seu projeto funcional. Ou seja, o web designer se preocupa com a aparência e com a funcionalidade de um website, pensando na navegabilidade e na interação que os usuários terão com a página da internet criada.
